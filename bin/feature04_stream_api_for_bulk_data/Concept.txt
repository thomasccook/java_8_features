Java Stream API for Bulk Data Operations on Collections

What:
	A new java.util.stream has been added in Java 8 to perform map/filter/reduce on Collections.
	Collection interface has been extended with stream() and parallelStream().
	
When:
	Java8+
	
Why: 
	We wish to find a brief syntax to process large data sets of Objects.
	
Notes:
	Underneath are the @FunctionalInterface, Function, BiFunction, Predicate, Consumer, etc. Classes.
